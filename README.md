# plaidkit

`plaidkit` is a Rust library in active development that aims to provide easy-to-use, ergonomic APIs for interacting with the Plaid service APIs. It is intended to make integrating Plaid into Rust applications straightforward and less error-prone. [web:5]

⚠️ **Important Notice**  
This version of the package is **not stable** and **not intended for public consumption**. APIs, functionality, and behavior may change without notice.  
**DO NOT USE THIS CRATE YET** in production or as a dependency.

## Status

- Current phase: early prototype / experimentation
- API surface: unstable
- Documentation: in progress

A stable release will be announced once the public API and behavior have been finalized. [web:5]
